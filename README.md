## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Abdullah-Junayed-290/Counter-App.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Counter-App
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```
